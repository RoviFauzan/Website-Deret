Selamat datang di repositori **Website Deret**! 🚀  
Proyek ini adalah sebuah platform berbasis web interaktif yang dirancang untuk membantu siswa dan pelajar memahami materi tentang deret angka, sekaligus menyediakan alat kalkulator pintar untuk menghitungnya secara instan.

## ✨ Fitur Utama

- 📖 **Materi Lengkap**: Penjelasan komprehensif mengenai berbagai jenis deret (seperti Deret Aritmatika dan Deret Geometri).
- 🖩 **Kalkulator Deret**: Alat bantu hitung otomatis untuk mencari suku ke-$n$ ($U_n$) dan jumlah $n$ suku pertama ($S_n$) dengan cepat dan akurat.
- 🎨 **Antarmuka Interaktif**: Desain UI/UX yang sederhana, responsif, dan mudah digunakan (User-Friendly).

## 🛠️ Teknologi yang Digunakan

Proyek ini dibangun menggunakan teknologi web dasar agar ringan dan mudah untuk dikembangkan:
- **HTML (94.5%)** - Struktur dasar halaman web dan konten materi.
- **CSS (3.0%)** - Gaya visual dan tata letak responsif.
- **JavaScript (2.5%)** - Logika interaktif untuk fungsi Kalkulator Deret.
